# Minimal macOS (CPU) requirements — for sync, plotting, node analysis, and
# code editing on a Mac. Install with:
#   pip install -r requirements_mac.txt
#
# This deliberately EXCLUDES the GPU/CUDA-only packages from requirements_core.txt
# that have no macOS wheels or require CUDA:
#   - torch/torchvision/torchaudio (+cu128) and the cu128 --extra-index-url
#   - kilosort (CUDA GPU sorter)
#   - cuda-python / cuda-bindings (pulled by spikeinterface[full])
#   - spikeinterface[full]  (the [full] extra drags in cuda-python)
# Run the full GPU pipeline (Trodes export, spike sorting, YOLO tracking) on the
# Windows machine with requirements_core.txt instead.
#
# External (non-pip) tool: join_views.py (stitching) shells out to ffmpeg —
# install it separately, e.g.  brew install ffmpeg
